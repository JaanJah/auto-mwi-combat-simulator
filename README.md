# auto-mwi-combat-simulator

```sh
# Install tampermonkey script
https://raw.githubusercontent.com/JaanJah/auto-mwi-combat-simulator/refs/heads/master/save_inventory.js
# Reload MWI, click on the "Copy Inventory to Clipboard" button,
# paste the copied data into the "input" directory and save it with `.json` extension
# e.g.
pbpaste > input/inventory.json
# Start the program
go run main.go
```
